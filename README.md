# Subhajit Bagchi | Full-Stack Developer & Founder

Welcome to my portfolio! This project is a high-impact, bold, and interactive showcase of my work in full-stack development, AI applications, and SaaS development. Built with **React 18**, **Framer Motion**, and **Vite**, this portfolio emphasizes a premium "compact and bounded" aesthetic with massive display typography and 3D interactions.

## 🚀 Key Features

- **3D Tilt Interaction**: Project cards that dynamically tilt and shift in 3D space based on your mouse movement, featuring a parallax "glassmorphism" depth for internal content.
- **Dynamic Typewriter Skills**: A custom-built, line-by-line typing and deleting animation that cycles through 6 categories of technical expertise (Languages, Frameworks, AI/ML, etc.).
- **LaunchNest Showcas**: A dedicated, interactive deep-dive into my SaaS pre-launch marketplace, featuring custom animations and a bold "tech stack" display.
- **Strobe Typography**: High-frequency strobing headers and responsive "clamp"-based typography for a raw, high-impact aesthetic.
- **Performance Optimized**: Built using pure CSS transitions and lightweight Framer Motion springs for high-performance visual effects.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Animations**: Framer Motion, Pure CSS, Custom Keyframes
- **Styling**: Vanilla CSS (Modern CSS3 with Variables, Clamp, and Grid)
- **Deployment Readiness**: Pre-configured for high-performance hosting environments.

## 🛠️ Local Development

To run the project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SubhajitBagchi03/portfolio.git
   cd portfolio-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
